This notebook contains example cells to download data.

Uses hard-wired lists of URLs for an example date/time, not easily adjusted. Full disk files are large. Executing this notebook is not necessary for the other notebooks.

To access Real-Time data without downloading, see the AWS_access_cookbook.

That cookbook could be adapted for archival data access of specified dates and times.

Another useful resource is https://satpy.readthedocs.io/en/latest/remote_reading.html

import requests
import os

def download_data(local_directory):
    # Ensure that the local directory exists; create it if it doesn't.
    os.makedirs(local_directory, exist_ok=True)

    # Iterate through the URLs and download files.
    for urld in urls2dwn:
        # Extract the filename from the URL.
        ntw = urld.split('/')[-1]

        # Construct the complete path to save the file in the local directory.
        file_path = os.path.join(local_directory, ntw)

        # Send an HTTP GET request to the URL.
        resp = requests.get(urld)

        # Check if the response is successful (status code 200).
        if resp.status_code == 200:
            # Write the content to the file in binary mode.
            with open(file_path, "wb") as file:
                file.write(resp.content)
            print(f"File '{ntw}' downloaded and saved to '{local_directory}'.")
        else:
            print(f"Failed to download '{ntw}' from the URL: {urld}")
